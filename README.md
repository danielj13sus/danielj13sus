# 🧑🏾‍💻 Daniel Jesus

**`Desenvolvedor Backend`**

Me chamo Daniel Jesus de Souza, tenho 21 anos e sou natural de Minas Gerais. Atualmente, estou cursando Engenharia de Software na Estácio. Desenvolvedor Backend apaixonado pelo ecossistema Java, focado em construir soluções corporativas eficientes, escaláveis e resilientes. Possuo uma base sólida em Java Core, Orientação a Objetos (POO) e Arquitetura de Software, aliada à experiência prática na implementação de lógicas complexas, programação funcional e tratamento de dados.

Atualmente, concentro meus estudos e projetos no desenvolvimento de APIs RESTful utilizando Spring Boot, aprofundando-me em Injeção de Dependências (IoC), gerenciamento de ciclo de vida de Beans, Spring MVC e configuração de ambientes (Spring Profiles). Além disso, integro essas aplicações a bancos de dados relacionais, utilizando PostgreSQL e linguagem SQL. 

Te convido a me acompanhar no LinkedIn, [Daniel Jesus - LinkedIn](https://www.linkedin.com/in/dev-daniel-jesus).
